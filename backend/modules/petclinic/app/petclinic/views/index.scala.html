Play REST API

This is a placeholder page to show you the REST API. Use httpie to post JSON to the application.

To see all posts, you can do a GET: 

    http GET localhost:9000/v1/posts

To create new posts, do a post 

    http POST localhost:9000/v1/posts title="Some title" body="Some Body"

Testdata

To insert testdata, do 

          http GET @petclinic.testdata.routes.InsertTestDataController.insertPetTypes().absoluteURL()
          http GET @petclinic.testdata.routes.InsertTestDataController.insertOwners().absoluteURL()
          http GET @petclinic.testdata.routes.InsertTestDataController.insertPets().absoluteURL()
          http GET @petclinic.testdata.routes.InsertTestDataController.insertSpecialties().absoluteURL()
